This is the simple version of the Web Redirector and Blocker extension. It has no user interface and is designed for maximum performance and efficiency. By using less code, it consumes fewer system resources.


KEY FEATURES

* No User Interface: This version operates in the background without any visible elements, providing a seamless experience.
* High Performance: With a minimalist design, it uses less processing power and memory.
* Lightweight Codebase: A smaller and more efficient code structure ensures faster operation.


USAGE

Once installed, the extension will start working automatically. There are no settings to configure.
